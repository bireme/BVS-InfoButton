1) If code is present, displayName MAY also be present, so that knowledge resources that do not support a given code system MAY still process the request using displayName as a search keyword.

2) If code is present, codeSystem SHALL be present.

4) If code is not present, the text displayed to (or entered by) the user SHALL be represented in the originalText attribute.
